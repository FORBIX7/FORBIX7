### 🚀 Избранные проекты

> **🤖 AI Telegram-бот для анализа баз данных**  
> *Дипломный проект: превращение естественного языка в SQL-запросы*  
> - **Задача:** Дать пользователю возможность получать аналитику из БД без знания синтаксиса SQL.  
> - **Решение:** Разработал асинхронного бота на `aiogram`, который принимает текстовый запрос, отправляет его в LLM вместе со схемой БД, получает валидный SQL-код, выполняет его и возвращает ответ (текст или ER-диаграмму).  
> - **Технологии:** `Python`, `aiogram`, `SQLite`, `OpenAI API / Local LLM`, `SQLAlchemy`.  
> - **Сложность:** Реализовал строгую валидацию и `try-except` блоки, чтобы бот не падал при галлюцинациях нейросети.

> **📝 Test Prep Platform (Веб-приложение)**  
> *Интерактивная платформа для подготовки к зачетам с алгоритмом интервального повторения*  
> - **Задача:** Заменить неудобные Word-файлы с тестами на удобный мобильный формат с умным обучением.  
> - **Решение:** Создал SPA на чистом `JavaScript`. Реализовал логику: неправильные ответы сохраняются и "прогоняются" снова до полного усвоения. Подключил `MathJax` для красивого рендеринга математических формул.  
> - **Результат:** Инструментом регулярно пользуются **125+ студентов**. Развернут и работает бесплатно через **GitHub Pages** (без бэкенда, данные подгружаются из репозитория).  
> - **Технологии:** `HTML5`, `CSS3`, `Vanilla JavaScript`, `MathJax`, `GitHub Pages`.

> **📦 WB Automation Script**  
> *Инструмент для автоматизации сборки и учета заказов Wildberries*  
> - **Задача:** Исключить человеческий фактор и рутину при обработке однотипных товаров из разных карточек.  
> - **Решение:** Написал скрипт на `Python`, который в реальном времени собирает данные о текущих заказах, кластеризует похожие позиции и автоматически списывает их с локальной базы данных `SQLite`.  
> - **Результат:** Процесс сборки ускорился, ошибки ручного ввода сведены к нулю.  
> - **Технологии:** `Python`, `SQLite`, `Requests` (или парсинг).
